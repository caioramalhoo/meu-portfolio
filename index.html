Eu sou Caio Ramalho

Atualmente estudo e pratico HTML5, CSS3 e JavaScript, avançando para projetos com React no front-end. No back-end, venho desenvolvendo APIs e lógica com Node.js, e utilizo Python para fortalecer minha base de programação e criar scripts/automação quando necessário. Meu foco é construir aplicações web completas e bem estruturadas, buscando unir uma boa experiência no front-end com uma base sólida no back-end.

Gosto de aprender na prática, então estou sempre criando projetos para consolidar conhecimento, melhorar organização do código, responsividade e boas práticas. Busco uma oportunidade de estágio/júnior onde eu possa evoluir em equipe, receber feedback, ganhar experiência com desafios reais e contribuir com comprometimento, curiosidade e constância.
